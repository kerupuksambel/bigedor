# BIGEDOR

BIGEDOR (**B**ig **I**ma**G**e **ED**it**OR**) is a web-based image editor built in React (Vite), and aimed to process high-resolution image.

## Features

### Editor

- [ ] Adding simple shapes
- [ ] Adding images
- [ ] Adding text
- [ ] Moving objects
- [ ] Object transformation
  - [ ] Rotation
  - [ ] Uniform translation
  - [ ] X and Y-axis translation

### Exporter

- [ ] Image exporting to PNG
- [ ] Scalable image exporting to PNG
